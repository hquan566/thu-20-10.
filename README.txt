
Tệp này chứa một trang web nhẹ "Thư 20/10" sẵn sàng để upload lên hosting tĩnh như GitHub Pages, Netlify, hoặc Vercel.
Các file trong gói:
- index.html
- style.css
- script.js
- photo.jpg (ảnh bạn cung cấp)

Hướng dẫn nhanh (GitHub Pages):
1. Tạo repository mới trên GitHub (ví dụ: 'thu-20-10').
2. Upload tất cả file trong gói vào repo (index.html, style.css, script.js, photo.jpg).
3. Vào Settings > Pages, chọn branch 'main' và folder '/' (root) -> Save. Sau vài phút, GitHub Pages sẽ cung cấp URL dạng: https://<your-username>.github.io/<repo-name> .
4. Khi bạn có URL chính thức, gửi cho mình link đó — mình sẽ tạo mã QR dẫn tới link và trả bạn ảnh QR đẹp sẵn sàng in/tặng.

Hoặc nếu muốn, bạn có thể upload folder này lên Netlify bằng tính năng 'Deploy from site' (drop folder) để có link ngay.
